# 🎮 Игра: “5 минут до провала”

##  Концепция

Игрок — оператор системы (реактор, станция, поезд и т.д.).  
Его задача — удерживать все системы в рабочем состоянии в течение 5 минут.

Если хотя бы одна система выходит из строя — игрок проигрывает.

---

#  ЭТАП 1 — БАЗА (MVP)


##  Основные элементы
- Таймер (5 минут)
- 3 системы:
  - Энергия
  - Охлаждение
  - Связь
- У каждой системы есть значение (0–100)

---
#  Действия игрока

## Кнопки:
- **1** → починить энергию (+20)
- **2** → починить охлаждение (+20)
- **3** → починить связь (+20)
- **стрелочки** → для передвижения

### НО:
 каждое действие тратит время / ресурс

---

# Ограничение действий

## Например:
- кулдаун 1 секунд  
- или “энергия игрока”

---

# Рандомные события

Каждые 10–20 сек:

- "Пожар!" → быстро падает энергия  
- "Перегрев!" → падает охлаждение  
- "Помехи!" → падает связь  

---

# Звук

- писк при критическом уровне  
- тревога  
- клик при починке  

---

# ЭТАП 4 — ПОЛИРОВКА

### 1. Критическое состояние

Если < 20:
- экран мигает  
- звук тревоги  

---

### 2. Разные режимы сложности

- **easy** → медленное падение
- **normal** → нормальное падение
- **hard** → всё ломается быстро  

---

### 3. Победа

Если 5 минут прошло:

 **"Система стабилизирована"**
